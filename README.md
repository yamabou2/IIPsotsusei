# IIPsotsusei

## 概要
職業訓練校の卒業制作として、
ストーリー性のあるゲームWebアプリケーションをチームで制作
担任からリーダーに任命され、実装だけでなく、仕様書およびプレゼンテーション資料の作成までを行い、
要件定義から成果物の説明までを一通り経験

## 使用技術
Java / Servlet & JSP / HTML / CSS など

## 機能一覧
・ログイン機能
・オセロゲーム
・Eゲーム
・ポーカー
・ジャンケンゲーム
・クイズゲーム

## 担当範囲
・Eゲーム実装
・仕様書・発表資料の作成


## 工夫した点
・データベース操作を用いて、ログイン時に使用したユーザー名をエンドロールで表示
・チーム開発を意識し、他メンバーが理解しやすいようにコードにコメントをつけた
・仕様書とプレゼン資料の作成に注力し、設計内容を言語化する経験を得ることを重視
・必要に応じてAIツールを活用し、詰まった部分や発展的な内容の部分の理解を深めながら実装を進めた
